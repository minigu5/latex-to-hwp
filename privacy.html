개인정보 처리방침 (Privacy Policy)

시행일: 2026년 5월 27일

1. 개인정보 수집 및 이용

휩 - LaTeX → 한글(HWP) 수식 변환기 (크롬 확장 프로그램 및 웹 서비스)는 사용자의 어떠한 개인정보나 데이터를 수집, 저장, 전송하지 않습니다.

모든 데이터 처리(수식 변환, 클립보드 읽기/쓰기, 이미지 OCR)는 오직 사용자의 브라우저(로컬 환경) 내에서만 수행됩니다. 외부 서버로 사용자의 데이터가 전송되거나 서버에 저장되는 일은 발생하지 않습니다.

2. 확장 프로그램 권한 사용 안내

크롬 확장 프로그램은 원활한 기능 제공을 위해 불가피하게 다음 권한을 요청하며, 명시된 목적 외에는 절대 사용되지 않습니다.

  • Clipboard (읽기/쓰기): 변환된 수식을 클립보드에 복사하거나, Gemini 등에서 복사된 수식을 읽어들여 변환하기 위해 임시 메모리에서만 사용됩니다.
  • Scripting / activeTab: 사용자가 팝업에서 명시적으로 '이 페이지에서 활성화'를 클릭했을 때에 한하여 해당 탭에 변환 스크립트를 주입하기 위해 사용됩니다.
  • Host Permissions: ChatGPT, Claude 등 지원되는 AI 챗봇 사이트에서 렌더링된 수식을 감지하고 변환 메뉴를 표시하기 위해 사용됩니다.

3. 제3자 데이터 공유

수집하는 어떠한 데이터도 없으므로, 사용자 정보를 제3자에게 제공하거나 판매, 공유하지 않습니다.

4. 문의처

개인정보 처리방침에 대한 문의 사항이나 확장 프로그램 관련 이슈는 GitHub Issues를 통해 남겨주시기 바랍니다.

휩 - LaTeX → 한글(HWP) 수식 변환기

AI가 만든 Markdown LaTeX 수식을 한컴오피스 한글 수식 편집기 문법으로 변환합니다. 결과를 복사해 한글 수식 편집기에 붙여넣으세요.

수식 이미지를 끌어다 놓거나

캡처 이미지를 붙여넣기(⌘/Ctrl+V)해도 됩니다. 첫 인식 때 AI 모델(약 80MB)을 한 번 내려받아 브라우저에 저장하며, 이후에는 바로 동작합니다. 이미지는 서버로 전송되지 않고 내 컴퓨터에서만 처리됩니다.

입력 (LaTeX)
출력 (한글 수식)
변환되면 자동으로 클립보드에 복사됩니다. 한글 수식 편집기에서 공백은 항 구분용이라 화면에 안 보이고, 백틱(`)은 좁은 간격을 의미합니다.
미리보기 (입력 수식 렌더링)
수식을 입력하면 여기에 렌더링됩니다.
변환 규칙은 한컴 공식 명세(한글 문서 파일 형식 - 수식, rev 1.2)를 참고했습니다 · GitHub
HWP / HWPX 파일 업로드
📂

HWP/HWPX 파일을 끌어다 놓거나

.hwp · .hwpx 지원 — 변환은 전부 브라우저에서 실행되며 서버로 전송되지 않습니다.

동작 방식
HWPX 파일은 ZIP+XML 구조로, 문서 안의 텍스트 LaTeX($…$, $$…$$, \(…\), \[…\])를 찾아 한컴 수식 개체(hp:equation)로 교체한 뒤 새 HWPX로 내보냅니다.
HWP(이진) 파일은 rhwp 오픈소스 엔진(인터넷 연결 필요)으로 HWPX 변환 후 같은 방식으로 처리합니다. — rhwp 엔진 로드 중, 잠시 기다려 주세요…
